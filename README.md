# BMO-Sketch
Etch-a-sketch with BMO.
[Live preview](https://lex-185.github.io/Etch-a-Sketch/)

Symbols:
* Blue triangle = Default pen 
* Green circle = Random pen
* Red circle = Select pen
* D-pad = Eraser
